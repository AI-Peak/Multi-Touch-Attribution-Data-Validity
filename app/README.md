# App Scaffold

This folder is a scaffold for the future interactive web application. No UI is implemented in this iteration.

The app should consume prepared JSON files from `app/data/generated/`, which are built from:

- SQL baseline outputs in `data/sql_outputs/`;
- Python analysis outputs in `analysis_python/outputs/`;
- model outputs in `model/*/outputs/`.

Planned interactive controls:

- benchmark threshold;
- attribution method selector;
- budget and revenue-per-conversion sliders;
- label scenario selector;
- scenario comparison panels.

The app should not train logistic regression or rebuild the Markov transition matrix inside the browser. Those heavier outputs are generated by the `model/` layer.
